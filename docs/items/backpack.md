# backpack

The backpack is an item that allows you to store items in it.

## Appearance

![backpack icon](/img/items/backpack.png)

## Details

* **Type:** Storage item
* **Stack size:** 1
* **placeable:** No

## Usage

The backpack can be used to store other items.
It can store up to 24 stacks of items.

## Crafting

Crafting a backpack requires a [crafting table](/docs/items/craft_table).

To craft a backpack, you will need the following materials:
* 10 [threads](/docs/items/thread)

You will have to do the following steps:
* [Thread weaving](/docs/crafting/thread_weaving) x3

One craft, will give you 1 backpack.
