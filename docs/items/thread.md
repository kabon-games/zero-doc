# thread

The thread is an item used in various crafting recipes.

## Appearance

![thread icon](/img/items/thread.png)

## Details

* **Type:** Resource
* **Stack size:** 200
* **placeable:** No

## Usage

Threads are used in crafting recipes.

## Drop

Threads can be obtained by harvesting contaminated leaves from trees using [silk worms](/docs/items/silk_worm).

## Versions

* Added in version 1.0
