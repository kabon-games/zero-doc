# wood hoe

The wood hoe is a basic farming tool used for tilling soil to prepare it for planting crops.

## Appearance

![wood hoe icon](/img/items/wood_hoe.png)

## Details

* **Type:** Tool
* **Stack size:** 1
* **placeable:** No

## Usage

The wood hoe can be used to till dirt blocks, turning them into farmland suitable for planting crops.

## Crafting

Crafting a wood hoe requires a [crafting table](/docs/items/craft_table).

To craft a wood hoe, you will need the following materials:
* 5 [wood](/docs/items/wood)

You will have to do the following steps:
* [sculpting](/docs/crafting/sculpting)

One craft will give you 1 wood hoe.
