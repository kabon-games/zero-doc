# sieve

The sieve is an item that allows you to filter materials from other items.
    
## Appearance

![sieve icon](/img/items/sieve.png)

## Details

* **Type:** Tool
* **Stack size:** 1
* **placeable:** Yes

## Usage

The sieve can be placed on the ground.

## Crafting

Crafting a sieve requires a [crafting table](/docs/items/craft_table).

To craft a sieve, you will need the following materials:
* 10 [wood](/docs/items/wood)
* 5 [threads](/docs/items/thread)

You will have to do the following steps:
* [nail hammering](/docs/crafting/nail_hammering)
* [thread weaving](/docs/crafting/thread_weaving)

One craft will give you 1 sieve.

## Block

### Appearance

![sieve block](/img/blocks/sieve.png)

### Details

* **Break level:** Hands
* **Resistance:** 1.5
* **Background:** No
* **Passthrough:** Yes

### Usage

The sieve block can be used to filter materials from other items.
Interact with the sieve with an item in hand to attempt to filter it and obtain other materials.
