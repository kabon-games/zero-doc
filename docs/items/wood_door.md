# wood door

The wood door is a basic door made of wood that can be used to secure buildings and structures.

## Appearance

![wood door icon](/img/items/wood_door.png)

## Details

* **Type:** Door
* **Stack size:** 20
* **placeable:** Yes

## Usage

The wood door can be placed to allow entry and exit from buildings.

## Crafting

Crafting a wood door requires a [crafting table](/docs/items/craft_table).

To craft a wood door, you will need the following materials:
* 4 [wood](/docs/items/wood)

There are no special crafting steps required.

One craft will give you 1 wood door.

## Block

### Appearance

![wood door block](/img/blocks/wood_door.png)

### Details

* **Break level:** Hands
* **Resistance:** 1.5
* **Background:** No
* **Passthrough:** Depends on state (open/closed)

### Usage

The wood door block can be opened and closed by interacting with it.
It provides a basic level of security for buildings.
