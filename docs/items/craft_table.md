# craft table

The craft table is an essential item that allows you to craft various other items in the game.

## Appearance

![craft table icon](/img/items/craft_table.png)

## Details

* **Type:** Crafting item
* **Stack size:** 20
* **placeable:** Yes

## Usage

The craft table can be placed on the ground and used to craft a variety of items.

## Crafting

To craft a craft table, you will need the following materials:
* 2 [wood](/docs/items/wood)

You will have to do the following steps:
* [nail hammering](/docs/crafting/nail_hammering) x3

One craft will give you 1 craft table.

### Appearance

### Details

* **Break level:** Hands
