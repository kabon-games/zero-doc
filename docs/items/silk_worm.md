# silk worm

The silk worm is used to produce threads for crafting.

## Appearance

![silk worm icon](/img/items/silk_worm.png)

## Details

* **Type:** consumable
* **Stack size:** 50
* **placeable:** No

## Usage

The silk worm can be used on a [tree](/docs/items/tree_seed) to obtain [threads](/docs/items/thread):
* Interact with the leaves of a tree while holding the silk worm to contaminate it.
* After some time, the contaminated leaves will become white and can be harvested for threads.
* Each harvest will yield 3-10 threads.
