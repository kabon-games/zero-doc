# wood pickaxe

The wood pickaxe is a basic mining tool used for breaking stone and ore blocks.

## Appearance

![wood pickaxe icon](/img/items/wood_pickaxe.png)

## Details

* **Type:** Tool
* **Stack size:** 1
* **placeable:** No

## Usage

The wood pickaxe can be used to break stone and ore blocks.

## Crafting

Crafting a wood pickaxe requires a [crafting table](/docs/items/craft_table).

To craft a wood pickaxe, you will need the following materials:
* 5 [wood](/docs/items/wood)

You will have to do the following steps:
* [sculpting](/docs/crafting/sculpting)

One craft will give you 1 wood pickaxe.
