# barrel

The barrel is an item that allows you to store fluids in it.

## Appearance

![barrel icon](/img/items/barrel.png)

## Details

* **Type:** Storage item
* **Stack size:** 20
* **placeable:** Yes

## Usage

The barrel can be placed on the ground.

## Crafting

Crafting a barrel requires a [crafting table](/docs/items/craft_table).

To craft a barrel, you will need the following materials:
* 5 [wood](/docs/items/wood)

You will have to do the following steps:
* [nail hammering](/docs/crafting/nail_hammering)

One craft, will give you 1 barrel.

## Block

### Appearance

![barrel block](/img/blocks/barrel.png)

### Details

* **Break level:** Hands
* **Resistance:** 1.5
* **Background:** No
* **Passthrough:** Yes

### Usage

The barrel block can be used to store fluids.
If the barrel is empty and it rains, it will fill up with water over time.
