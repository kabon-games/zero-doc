# wood bucket

The wood bucket is an item that allows you to carry fluids in it.

## Appearance

![wood bucket icon](/img/items/wood_bucket.png)

## Details

* **Type:** Fluid container
* **Stack size:** 1
* **placeable:** No

## Usage

The wood bucket can be used to carry fluids such as water.

## Crafting

Crafting a wood bucket requires a [crafting table](/docs/items/craft_table).

To craft a wood bucket, you will need the following materials:
* 5 [wood](/docs/items/wood)

You will have to do the following steps:
* [nail hammering](/docs/crafting/nail_hammering)

One craft will give you 1 wood bucket.
