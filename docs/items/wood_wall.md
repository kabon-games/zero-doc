# wood wall

The wood wall is a basic building block used for constructing structures and providing shelter.

## Appearance

![wood wall icon](/img/items/wood_wall.png)

## Details

* **Type:** Building Block
* **Stack size:** 200
* **placeable:** Yes

## Usage

The wood wall can be placed to create walls for buildings and shelters.

## Crafting

Crafting a wood wall requires a [crafting table](/docs/items/craft_table).

To craft a wood wall, you will need the following materials:
* 1 [wood](/docs/items/wood)

There are no special crafting steps required.

One craft will give you 1 wood walls.
