# tree seed

The tree seed is used to plant new trees.

## Appearance

![tree seed icon](/img/items/tree_seed.png)

## Details

* **Type:** Plantable item
* **Stack size:** 100
* **placeable:** Yes

## Usage

The tree seed can be planted on dirt or grass blocks to grow a new tree.

## Blocks

### Appearance

![tree seed block](/img/blocks/tree.png)

### Details

#### sapling

* **Break level:** Hands
* **Resistance:** 0.5
* **Background:** No
* **Passthrough:** Yes

#### grown tree

* **Break level:** wood axe
* **Resistance:** 3
* **Background:** No
* **Passthrough:** Yes

### Usage

When planted, the tree seed will grow into a tree.
The tree take at least 10 seconds to grow from a sapling to a fully grown tree.

When broken, the grown tree will drop :
* 0-2 [tree seed](/docs/items/tree_seed)
* 3-10 [wood](/docs/items/wood)
* 0-1 [silk worm](/docs/items/silk_worm)

If the tree is contaminated by a [silk worm](/docs/items/silk_worm), it might also drop 3-10 [thread](/docs/items/thread).
